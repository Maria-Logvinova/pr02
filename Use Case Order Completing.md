# Сценарий: Оформление заказа

**Цель**: Оформить заказ на сайте, чтобы получить свой заказ

## Основной сценарий:
1. Пользователь запускает оформление заказа из экрана корзины
2. Сайт выводит экран для оформления заказа
3. Экран оформления заказа отображает поля для выбора типа адреса, ввода адреса, выбора даты и времени заказа, ввода промокода, переключатель позвонить ли для подтверждения
4. Пользователь выбирает тип и вводит адрес
5. Экран подтверждает успешный выбор адреса
6. Пользователь устанавливает дату и время доставки
7. Пользователь вводит промокод
8. Экран подтверждает успешное применение промокода
9. Пользователь указывает, нужно ли звонить для подтверждения
10. Экран отображает итоговую стоимость заказа с учетом скидки, а также стоимость товаров и доставки
11. Экран показывает возможность перехода пользователя к оплате

## Альтернативные сценарии:
**Пользователь не авторизован:**  
&nbsp;&nbsp;&nbsp;&nbsp;2.1\. Сайт выводит экран авторизации  
&nbsp;&nbsp;&nbsp;&nbsp;2.2\. Пользователь проходит авторизацию любым доступным способом  
&nbsp;&nbsp;&nbsp;&nbsp;2.3\. Сайт подтверждает успешную авторизацию  
&nbsp;&nbsp;&nbsp;&nbsp;2.4\. Сайт выводит экран для оформления заказа  
&nbsp;&nbsp;&nbsp;&nbsp;3\. Возврат на [шаг 3](#экран-оформления-заказа-отображает-поля-для-выбора-типа-адреса,-ввода-адреса,-выбора-даты-и-времени-заказа,-ввода-промокода,-переключатель-позвонить-ли-для-подтверждения) основного сценария

**Пользователь захотел поменять состав заказа:**  
&nbsp;&nbsp;&nbsp;&nbsp;3\. Сайт оформления заказа помимо полей для оформления заказа отображает кнопку возврата к корзине  
&nbsp;&nbsp;&nbsp;&nbsp;4\. На любом из шагов 4-10 основного сценария пользователь возвращается в корзину, изменяет состав пицц и/или топпингов  
&nbsp;&nbsp;&nbsp;&nbsp;5\. Пользователь переходит на экран оформления заказа  
&nbsp;&nbsp;&nbsp;&nbsp;6\. Возврат к тому же шагу основного сценария, с которого пользователь вернулся в корзину  

## Исключения:
**Адрес введен неверно:**  
&nbsp;&nbsp;&nbsp;&nbsp;5\. Экран выводит ошибку ввода адреса

**Применен некорректный промокод:**  
&nbsp;&nbsp;&nbsp;&nbsp;8\. Экран выводит ошибку о несуществующем промокоде
